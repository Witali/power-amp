# TODO: улучшение OCR Tesseract для годовых оглавлений

Текущий результат на оглавлениях `Радио` за 1995-2000 годы:

- 1626 записей всего
- 998 строк статей с номером выпуска и страницей
- 504 строки `needs_review`

Изменение оставляем в пайплайне, если оно дает ощутимый выигрыш:

- заметно увеличивает число `article`-строк без роста явного мусора;
- или снижает число `needs_review`;
- или исправляет устойчивый класс ошибок на нескольких страницах;
- и проходит существующие тесты extractor/layout.

  правым числовым столбцом.

- [ ] Подбирать PSM по типу блока: отдельные настройки для heading, обычных

- [ ] Проверить предобработку под бумагу: deskew, локальная бинаризация,

- [ ] Собрать маленький золотой набор вручную вычитанных строк/страниц и

- [ ] Рассматривать дообучение Tesseract на шрифтах журнала только после
