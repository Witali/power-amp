# Study Materials

This folder contains research notes, OCR studies, source summaries, topology comparisons, calibration reports and exploratory simulations.

Use this folder for material that helps design or understand circuits, but is not itself a finalized result package. Finished generated schematics, plots and per-circuit reports should stay under `results/`.

Current studies:

- `amp_no_feedback_study/`: Radio magazine amplifier research, no-global-feedback amplifier notes, source comparisons, OCR workflow notes and amplifier topology studies.
