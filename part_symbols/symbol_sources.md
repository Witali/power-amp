## Notes For This Project

- Prefer official standards or publisher pages when available.
- Use GOST/ESKD-style UGO for Russian radio schematics.
- Use IEC/IEEE awareness when reading English-language magazines, because older American schematics often use zig-zag resistors while European/Russian schematics usually use rectangular resistors.
- Do not copy standard drawings directly into generated result schematics. Redraw clean, project-local symbols with stable proportions.
